# resumesite
My resume/CV website, about my personal information, educational background and experiences. To see the website: https://halukoz.github.io/resumesite/

##Technologies
HTML-5, CSS3, Javascript. 
This is a responsive website (mobile and desktop application) and multilingual (english and german).

## How to install
* You do not need to install any program, just open the index.html file with a browser.
